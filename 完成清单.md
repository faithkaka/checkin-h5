## 🔧 故障排查

### 问题 1: 用户 ID 为 guest_xxx
**原因**: 未在支付宝环境打开，也没有传入 userId 参数

**解决**: 添加 URL 参数
```
http://localhost:9000/index.html?userId=test_user_001
```

### 问题 2: 数据没有保存
**检查**:
1. 浏览器控制台是否有 Supabase 错误
2. 检查 users 表是否有记录
3. 查看 Supabase Logs

### 问题 3: 字段不存在错误
**解决**: 确保已执行 `supabase-init.sql`，使用 `alipay_user_id` 而不是`user_id`
