## 📊 功能模块

### 1. 数据概览（Dashboard）

**实时显示**：
- 📈 总用户数
- 📍 今日打卡数
- 📊 总打卡次数
- 🏆 最高积分用户

**TOP 10 排行榜**：
- 显示积分最高的 10 个用户
- 包含用户 ID、积分、打卡数、注册时间

### 2. 用户管理

**功能**：
- 👥 查看所有用户列表
- 📊 查看每个用户的积分和打卡数
- 🗑️ 删除用户（包含其所有打卡记录）

**操作**：
1. 进入"用户管理"
2. 查看用户列表
3. 点击"删除"按钮可删除用户

### 3. 打卡记录

**功能**：
- 📍 查看所有打卡记录
- 🔍 最近 100 条打卡
- 📅 显示打卡时间

**信息展示**：
- 用户 ID
- 打卡点 ID
- 获得积分
- 打卡时间

### 4. 打卡点管理

**功能**：
- 🗺️ 添加新打卡点
- ✏️ 编辑打卡点信息
- 🗑️ 删除打卡点

**打卡点字段**：
- **ID**: 打卡点唯一标识（1-100）
- **名称**: 景点名称（如：解放碑步行街）
- **图标**: Emoji 图标（如：🏢）
- **积分**: 打卡获得的积分
- **地址**: 详细地址
- **时间范围**: 建议打卡时间（如：8:00-9:30）
- **描述**: 景点描述

**添加打卡点步骤**：
1. 点击"➕ 添加打卡点"
2. 填写表单信息
3. 点击"保存"
4. 刷新页面查看

### 5. 成就管理

**功能**：
- 🏆 查看所有成就配置
- 📊 查看 achievement 信息

**成就字段**：
- ID、名称、所需积分、图标、描述

## 📊 SQL 查询示例

在管理后台控制台可以运行：

```javascript
// 查询今日新增用户
const today = new Date().toISOString().split('T')[0];
const { data } = await supabase
  .from('users')
  .select('*')
  .gte('created_at', today);

// 查询某用户的所有打卡
const { data } = await supabase
  .from('checkins')
  .select('*')
  .eq('alipay_user_id', 'alipay_xxx');

// 统计每个打卡点的打卡次数
const { data } = await supabase
  .from('checkins')
  .select('checkpoint_id');
  
const count = {};
data.forEach(c => {
  count[c.checkpoint_id] = (count[c.checkpoint_id] || 0) + 1;
});
console.table(count);
```
