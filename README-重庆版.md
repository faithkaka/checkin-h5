## 📋 打卡规则

### 积分规则
1. 每个必打卡点打卡成功获得对应积分
2. 推荐点位自愿打卡，每个 5 分
3. 积分可累计，用于达成不同等级成就

### 兑奖规则
1. ⭐ **达成任意成就后，只能兑奖一次**
2. 已兑奖的成就标记为"已兑奖"
3. 其他成就显示"已达兑奖上限"
4. 兑奖时需向工作人员出示兑奖码

### 打卡流程
1. 打开首页查看地图
2. 点击地图上的打卡点标记
3. 打卡成功后获得积分
4. 积分达到条件自动解锁成就
5. 前往奖品页查看成就卡片
6. 点击"线下兑奖"生成兑奖码

## 🛠️ 使用方式

### 本地预览
```bash
# 方式 1: 直接打开
open checkin-h5/index.html

# 方式 2: 启动本地服务器
cd checkin-h5
python3 -m http.server 8080
```

### 手机预览
确保手机和电脑在同一网络：
```bash
# 查看本机 IP
ifconfig | grep "inet "

# 手机访问 http://<你的IP>:8080
```
