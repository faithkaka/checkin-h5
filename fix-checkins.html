🔧 修复 checkins 表

📋 问题诊断

checkins 表缺少 checked_at 字段，导致管理后台无法加载打卡记录。

解决方案：在 Supabase 后台添加该字段

📝 方法 1：使用 Supabase 界面（推荐）

  1. 打开 Supabase Dashboard
  2. 进入 Table Editor
  3. 选择 checkins 表
  4. 点击 Add column
  5. 填写：
    • Name: checked_at
    • Type: timestamptz
    • Default: now()
    • Nullable: 不勾选
  6. 点击 Save

💻 方法 2：使用 SQL（更快）

在 Supabase 的 SQL Editor 中运行以下 SQL：

ALTER TABLE checkins 
ADD COLUMN checked_at timestamptz DEFAULT now() NOT NULL;

-- 更新现有数据
UPDATE checkins 
SET checked_at = created_at 
WHERE checked_at IS NULL;
