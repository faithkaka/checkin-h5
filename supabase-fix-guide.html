🔧 Supabase SQL 修复指南

📋 问题

checkins 表缺少 checked_at 字段，导致管理后台无法加载打卡记录。

🚀 操作步骤

  1. 打开 Supabase Dashboard
    👉 点击这里直接进入 SQL Editor
  2. 点击 "New Query" 按钮
  3. 复制下面的 SQL 代码
  4. 粘贴到 SQL Editor
  5. 点击 "Run" 按钮执行
  6. 看到 "Success. No rows returned" 说明执行成功

💻 SQL 代码

-- 1. 添加 checked_at 字段到 checkins 表
ALTER TABLE checkins 
ADD COLUMN checked_at timestamptz DEFAULT now() NOT NULL;

-- 2. 为现有数据设置 checked_at 值（使用 created_at）
UPDATE checkins 
SET checked_at = created_at 
WHERE checked_at IS NULL;

-- 3. 验证字段已添加
SELECT column_name, data_type, column_default, is_nullable
FROM information_schema.columns
WHERE table_name = 'checkins'
ORDER BY ordinal_position;

✅ 已复制到剪贴板！

✅ SQL 说明

第 1 条：添加 checked_at 字段，类型为时间戳（带时区），默认值为当前时间

第 2 条：为现有数据填充 checked_at 值（使用 created_at）

第 3 条：验证字段是否成功添加
