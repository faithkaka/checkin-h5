## 🎨 动画效果

### 页面切换（slideIn）
```css
@keyframes slideIn {
  from { opacity: 0; transform: translateX(20px); }
  to { opacity: 1; transform: translateX(0); }
}
```
- 持续时间：0.25s
- 效果：页面从右侧滑入 + 淡入

### 导航图标弹跳（navBounce）
```css
@keyframes navBounce {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.3); }
}
```
- 持续时间：0.4s
- 效果：图标先放大到 1.3 倍，然后恢复

### 点击缩放
- 点击时：scale(0.95) - 轻微缩小
- 释放后：立即恢复
- 持续时间：0.1s
